Decision Simulator


Project Description:

Decision Simulator is a lightweight web application that helps users make decisions through interactive simulation logic.
It demonstrates frontend logic handling and UI interaction without frameworks.


Tech Stack / Built With:

- HTML5
- CSS3
- JavaScript (Vanilla)
- Web Audio API


Features:

- Interactive decision generation system
- Dynamic UI updates based on user input
- Sound feedback using Web Audio API
- Fully dependency-free implementation


Getting Started / Installation:

git clone https://github.com/danemajstorcev/decision-simulator.git
- cd decision-simulator
- open index.html
- open LiveServer
